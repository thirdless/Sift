5. Modulele

Ce este un modul?

Luați în considerare un modul ca același lucru cu o bibliotecă de coduri.

Un fișier care conține un set de funcții pe care doriți să le includeți în aplicația dvs.

Există mai multe module încorporate în Python, pe care le puteți importa oricând doriți.


Python are un modul numit timp pentru a gestiona sarcinile legate de timp. Pentru a utiliza funcțiile definite în modul, trebuie mai întâi să importăm modulul.

import time

Funcția time() returnează numărul de secunde trecute de la epocă.

Pentru sistemul Unix, 1 ianuarie 1970, 00:00:00 la UTC este epoca (punctul în care începe timpul).

import time
secunde = time.time()
print("Secunde trecute de la epoca =", secunde)

Modulul matematic este un modul standard în Python și este întotdeauna disponibil. Pentru a utiliza funcții matematice sub acest modul, trebuie să importați modulul folosind import math.

import math
print(math.sqrt(4))  # va afisa valoarea 2, rezultatul radicalului lui 4

Cerință

Să se afișeze toți radicalii numerelor de la 1 la 30, folosind o buclă for și modulul math. De asemenea se va afișa ora, minutul și secunda înainte de începerea operațiilor, dar și după finalul acestora.
