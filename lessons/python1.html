1. Introducere în Python

Ce este Python?

Python este un limbaj de programare dinamic multi-paradigmă, creat în 1989 de programatorul olandez Guido van Rossum.

Python este unul dintre cele mai folosite limbaje, și presupune opțiunea programatorilor datorită simplității sintaxei, ale variabilelor dinamice și rapidității, deși este totuși interpretat.

Pentru a afișa text pe ecran, se folosește instrucțiunea print, iar textul dinăuntrul funcției trebuie pus între ghilimele pentru a marca faptul că este definită o bucată de text, ci nu o variabilă sau altfel de instrucțiune; de asemenea se va pune parametrul dorit în paranteze pentru a putea apela funcția cu parametrul dorit:

print("Text pus între ghilimele")

Funcția de mai sus va afișa textul Text pus între ghilimele

Cerință

Afișează pe ecran textul "Hello world!", folosind funcția print
