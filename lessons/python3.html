3. Funcțiile

O funcție este un bloc de cod care rulează numai atunci când este apelat.

Puteți trece date, cunoscute sub numele de parametri, într-o funcție.

O funcție poate returna date ca rezultat.

Crearea unei funcții

În Python o funcție este definită folosind cuvântul cheie def:

def my_function():
    print("Hello from a function")

Apelarea unei funcții

Pentru a apela o funcție, utilizați numele funcției urmat de paranteză:

def my_function():
    print("Hello from a function")
  
my_function()

Cerință

Definește o funcție numită prima_functie ce va avea doi parametri, a și b, iar înăuntrul funcției se va efectua adunarea celor două variabile, ca în lecția prcedentă. Funcția va fi apelată cu doi parametri la alegere, după care se va afișa pe ecran rezultatul.
