4. Buclele

Bucla for

O buclă for este utilizată pentru iterarea unei secvențe (adică o listă, un tuplu, un dicționar, un set sau un șir).

Acest lucru este mai puțin asemănător cuvântului cheie for din alte limbaje de programare și funcționează mai mult ca o metodă de iterare așa cum se găsește în alte limbaje de programare orientate pe obiecte.

Cu bucla for putem executa un set de instrucțiuni, o dată pentru fiecare element dintr-o listă, tuplu, set etc.

fruits = ["apple", "banana", "cherry"]
for x in fruits:
    print(x)

Bucla for nu necesită o variabilă de indexare pentru a fi setată în prealabil.

Declarația break

Cu instrucțiunea break putem opri bucla înainte ca aceasta să parcurgă toate elementele: 

fruits = ["apple", "banana", "cherry"]
for x in fruits:
    if x == "banana":
        break
    print(x)

Cerință

Iterează prin șirul de caractere "Ana are mere, nu are pere" și reține de câte ori apare litera a în secvența respectivă, după care afișează rezultatul.
